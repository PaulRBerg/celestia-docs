# Leap integration with Celestia

This tutorial will go over how you can add Celestia network parameters to Leap wallet.

## Install Leap

Leap is a popular Cosmos-based wallet that allows anyone
to connect to Tendermint chains from their browser or phone.

Learn more and download Leap [here](https://www.leapwallet.io/).
